# E2Eテスト用フィクスチャ

このディレクトリにはE2Eテストで使用するテスト用ファイルを配置します。

## 必要なファイル

### 画像ファイル

| ファイル名 | 説明 | 要件 |
|------------|------|------|
| `valid-face.jpg` | 正常な顔画像 | JPEG形式、正面を向いた顔、10MB以下 |
| `valid-face.png` | 正常な顔画像（PNG） | PNG形式、正面を向いた顔、10MB以下 |

### バリデーションテスト用（オプション）

| ファイル名 | 説明 | 要件 |
|------------|------|------|
| `image.gif` | 不正な形式テスト用 | GIF形式の画像 |
| `large-file.jpg` | サイズ超過テスト用 | 10MB超のJPEG画像 |
| `landscape.jpg` | 顔未検出テスト用 | 顔が写っていない風景写真 |
| `group-photo.jpg` | 複数顔テスト用 | 複数人が写った集合写真 |

## 画像の作成方法

### valid-face.jpg の作成例

1. 正面を向いた顔写真を用意
2. JPEG形式で保存（品質80%程度推奨）
3. ファイルサイズが10MB以下であることを確認

### テスト用画像の生成スクリプト（例）

```bash
# ImageMagickを使用して1x1のダミー画像を作成
convert -size 100x100 xc:gray valid-face.jpg
```

## 注意事項

- テスト用画像はGitにコミットしないことを推奨（.gitignoreに追加）
- CI環境では画像を動的に生成するか、事前にアップロードしておく
- 個人の顔写真を使用する場合はプライバシーに注意
